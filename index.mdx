Broń Vault is an open-source stealer logs parser and visualization dashboard built for security analysts. Drop a `.zip` log file into the web interface and the application automatically structures its contents — credentials, host information, software inventory, and files — ready for immediate inspection and cross-log correlation.

## Get started in minutes

<Steps>
  <Step title="Clone and configure">
    Clone the repository and copy `.env.example` to `.env`, setting secure passwords for MySQL, ClickHouse, and your JWT secret.
  </Step>
  <Step title="Start all services">
    Run `bash docker-start.sh` to bring up MySQL, ClickHouse, MinIO, and the Next.js application with a single command.
  </Step>
  <Step title="Log in">
    Open `http://localhost:3000` and sign in with the default admin credentials. Change the password immediately after first login.
  </Step>
  <Step title="Upload your first log">
    Navigate to **Upload**, drag and drop a stealer log `.zip` file, and watch real-time progress as the parser structures the data.
  </Step>
</Steps>
